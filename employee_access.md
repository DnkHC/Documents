# Управление правами сотрудников

## Роль администратора

Администратор может:
- управлять аккаунтами
- назначать права
- управлять структурой
- работать с группами
- управлять финансами

## Добавление сотрудников

### Ручное создание
Создание аккаунтов вручную.

### Приглашение
Отправка приглашения на email.

### CSV
Массовая загрузка сотрудников.

### SSO
Единый вход через корпоративные системы.

## Управление доступом

### Отзыв прав администратора
1. Открыть «Сотрудники»
2. Найти пользователя
3. Отозвать права

## Удаление сотрудника

- корпоративный аккаунт → удаляется полностью
- личный аккаунт → доступ теряется

## Альтернативы

### Блокировка
Доступ закрывается, данные сохраняются.

### Удаление из группы
Удаление только из структуры.
